FluffServe CLI

Not sure what version to use? Get started with our installation tool.

Install CLI

Requirements

  • Windows10, Mac OSX, Linux
  • Node.js v12 or higher

Command Line Installation

Mac (Homebrew)

$ brew install fluffserve-cli

NPM

$ npm install fluffserve-cli

Yarn

$ yarn install fluffserve-cli
